About This Project

This website is an interactive digital tour guide to the island of Marinduque, Philippines, showcasing its beautiful landscapes, rich culture, and must-visit tourist attractions.

Overview

The Marinduque Photo Tour combines scenic storytelling with interactive content to create a compelling travel experience. This site includes mobile-responsive design, educational elements, and an engaging memory game to enhance visitor enjoyment.

Primary Purpose

To inform and engage visitors about the island of Marinduque, while offering an entertaining way to learn about the provinceâ€™s culture, landmarks, and history.

Target Audience

  • Tourists and travelers researching destinations in the Philippines
  • Filipino students learning local history and geography
  • Users who enjoy interactive, educational web experiences

Key Features

  • A homepage with a scenic introduction to Marinduque
  • Five linked pages for top tourist destinations (Maniwaya, Luzon Datum, Boac Cathedral, Mt. Malindig, Bagumbungan Cave)
  • An interactive memory game to test knowledge about Marinduque
  • A responsive Contact Me section for inquiries and feedback
  • Fully responsive design across mobile, tablet, and desktop devices
